## 问题回答

    - 在完成代码的过程中几乎没有遇到任何问题，只需看懂逻辑，调用已经实现好的绘制函数即可。
    - 在完成代码后，图片在窗口中正常显示，但是保存后图片为全黑
        - 我多次尝试了重装OpenGL库，但是没有变化；因为编译安装较为复杂，所以没有尝试
